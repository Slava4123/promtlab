Напиши оптимизированный SQL-запрос для PostgreSQL. Найди пользователей с наибольшим количеством промптов за последние 30 дней
SQL-запрос для анализа активности:

SELECT u.username, u.email,
  COUNT(p.id) AS prompt_count,
  MAX(p.created_at) AS last_created
FROM users u
JOIN prompts p ON p.user_id = u.id
WHERE p.created_at >= NOW() - INTERVAL '30 days'
  AND p.deleted_at IS NULL
GROUP BY u.id, u.username, u.email
ORDER BY prompt_count DESC
LIMIT 20;
