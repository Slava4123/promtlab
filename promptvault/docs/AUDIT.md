# Аудит PromptVault — 2026-04-06

## Критичные

1. **Goroutine leak в email** — письма теряются при рестарте сервера
2. **Rate limit forgot-password** — нет защиты от спама кодами сброса + timing-атака
3. **`git reset --hard` в деплое** — уничтожает любые локальные изменения на VPS
4. **Evict-горутина rate limiter** — не останавливается, утечка при тестах

## Архитектура

5. **Email-сервис как интерфейс** — единственное нарушение DIP в проекте
6. **Worker pool для фоновых задач** — формализовать 6 `go func()` в единый паттерн
7. **OpenRouter URL/timeout в конфиг** — сейчас захардкожены

## Производительность

8. **Индекс `collections.team_id`** — sequential scan на командных коллекциях
9. **JOIN вместо subquery COUNT** — в списке команд (`team_repo.go:60`)
10. **Code splitting фронтенда** — React.lazy по маршрутам, сейчас всё в одном бандле
11. **Optimistic update на favorite** — убрать полный refetch при toggle

## Качество кода

12. **Фронтенд-тесты** — 2 файла vs 19 на бекенде, инфраструктура уже готова
13. **Accessibility** — aria-live, skip-to-content, фокус в диалогах
14. **CSS-переменные в AI markdown** — захардкоженные цвета не адаптируются к теме
15. **Slug collision** — проверять тип ошибки + увеличить суффикс

## Новые фичи

16. **Подписки (YooKassa)** — Free/Pro/Max, основа монетизации
17. **Open Graph теги** — превью при шаринге в Telegram/VK
18. **Email при приглашении в команду** — сейчас пользователь не узнаёт о приглашении
19. **Экспорт/импорт промптов** — data portability для self-hosted аудитории

## Инфраструктура

20. **Мониторинг** — сейчас только health check + slog
21. **Security scanning в CI** — govulncheck + npm audit
22. **Бэкапы БД** — нет стратегии, потеря данных при сбое
23. **Docker image registry** — собирать образы в CI, не на VPS
